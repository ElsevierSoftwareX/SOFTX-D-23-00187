EPCIS Capture Service
Method Name Description
Event Capture Description: This service allows you to store your EPCIS Document into your backend storage (e.g. Mongo DB)
Method: post
Contents: raw-xml-text, complying with EPCIS v1.1 XML Schema
Example URL: http://{base-url}:{base-port}/epcis/Service/eventCapture
Example EPCIS Document: Example of EPCISDocument
Vocabulary Capture Description: This service allows you to store your EPCIS Master Data Document into your backend storage (e.g. Mongo DB)
Method: post
Contents: raw-xml-text, complying with EPCIS v1.1 XML Schema
Example URL: http://{base-url}:{base-port}/epcis/Service/vocabularyCapture

EPCIS Query Service
Method Name Description
GetStandardVersion Description: Return the standard version of EPCIS
URL: http://{base-url}:{base-port}/epcis/Service/query/StandardVersion
Additional Parameters: Null
GetVendorVersion Description: Return the vendor specific version of EPCIS
URL: http://{base-url}:{base-port}/epcis/Service/query/VendorVersion
Additional Parameters: Null
GetQueryNames Description: Return a list of all query names available
URL: http://{base-url}:{base-port}/epcis/Service/query/QueryNames
Additional Parameters: Null
Return: [ {queryNames} ]
Poll Description: Return events stored in repository in interest immediately
URL: http://{base-url}:{base-port}/epcis/Service/query//Poll/{queryName}
Additional Parameters:
eventType : Comma separated event type (e.g. ObjectEvent)
(GE|LT)_eventTime : yyyy-MM-dd'T'HH:mm:ss.SSSXXX
(GE|LT)_recordTime : yyyy-MM-dd'T'HH:mm:ss.SSSXXX
EQ_action : Comma separated action (e.g. ADD)
EQ_bizStep : Comma separated bizStep
EQ_disposition : Comma separated disposition
(EQ|WD)_readPoint : Comma separated readPoint
(EQ|WD)_bizLocation : Comma separated bizLocation
EQ_transformationID : Comma separated transformationID
MATCH_(epc|parentID|inputEPC|outputEPC|anyEPC) : Comma separated EPC
MATCH_(epcClass|inputEPCClass|outputEPCClass|anyEPCClass) : Comma separated EPC Class
orderBy : Field used by sorting (e.g. eventTime)
orderDirection : ASC, DESC (default)
eventCountLimit : return specified number of events
maxEventCount : yield error if # of events exceeds specified number
vocabularyName : Comma separated vocabularyName
includeAttributes : whether including attributes or not
includeChildren : whether including children or not
attributeNames : Comma separated attribute names, result only will include specified list of names
(EQ|WD)_name : Comma separated attribute names
HASATTR : Comma separated attribute names
maxElementCount : yield error if # of events exceeds specified number
GetSubscriptionIDs Description: Returns a list of all subscriptionIDs currently subscribed to the specified named query.
URL: http://{base-url}:{base-port}/epcis/Service/query/SubscriptionIDs/{queryName}
Additional Parameters: Null
Return: [ {SubscriptionIDs} ]
Subscribe Description: Registers a subscriber for a previously defined query having the specified name.
URL: http://{base-url}:{base-port}/epcis/Service/query/Subscribe/{queryName}/{subscriptionID}
Additional Parameters:
dest : Callback URL
cronExpression : Quartz CronTrigger Tutorial
Unsubscribe Description: Removes a previously registered subscription having the specified subscriptionID.
URL: http://{base-url}:{base-port}/epcis/Service/query/Unsubscribe/{subscriptionID}
Additional Parameters: null
