EPCIS Query Service

Method Name Description
Poll
  • Return events stored in repository in interest immediately
  • URL: http://{base-url}:{base-port}/epcis/Service/Poll/{queryName}
  • Additional Parameters:

    {queryName}==SimpleEventQuery

    eventType : CSV, Regex (e.g., ObjectEvent, (.)*Event^regex)
    (GE|LT)_eventTime : yyyy-MM-dd'T'HH:mm:ss.SSSXXX
    (GE|LT)_recordTime : yyyy-MM-dd'T'HH:mm:ss.SSSXXX
    (GE|LT)_errorDeclaration Time : yyyy-MM-dd'T'HH:mm:ss.SSSXXX
    EQ_action : CSV (e.g. ADD, OBSERVE, DELETE)
    EQ_bizStep : CSV, Regex (e.g., urn:epcglobal:cbv:bizstep:shipping, urn:epcglobal:cbv:bizstep:........^regex)
    EQ_disposition : CSV, Regex (e.g., urn:epcglobal:cbv:disp:in_transit, urn:epcglobal:cbv:disp:..........^regex )
    (EQ|WD)_readPoint : CSV, Regex (e.g., urn:epc:id:sgln:0614141.07346.1234 urn:epc:id:sgln:0614141[.]07346[.]....^regex )
    (EQ|WD)_bizLocation : CSV, Regex (e.g., urn:epc:id:sgln:0614141.07346.1234 urn:epc:id:sgln:0614141[.]07346[.]....^regex )
    EQ_transformationID : CSV, Regex
    MATCH_(epc|parentID|inputEPC|outputEPC|anyEPC) : CSV, Regex (e.g., urn:epc:id:sgtin:0614141.107346.2018, urn:epc:id:sgtin:0614141[.]......[.]....^regex )
    MATCH_(epcClass|inputEPCClass|outputEPCClass|anyEPCClass) : CSV, Regex (e.g., urn:epc:class:lgtin:4012345.012345.998877, urn:epc:idpat:sgtin:4012345.098765.*, urn:epc:class:lgtin:4012345[.]012345[.]......^regex )
    EQ_bizTransaction_type : CSV (e.g., EQ_bizTransaction_urn:epcglobal:cbv:btt:po = http://transaction.acme.com/po/12345678 )
    EQ_source_type : CSV (e.g., EQ_source_urn:epcglobal:cbv:sdt:possessing_party = urn:epc:id:sgln:4012345.00001.0)
    EQ_destination_type : CSV (e.g., EQ_destination_urn:epcglobal:cbv:sdt:owning_party = urn:epc:id:sgln:0614141.00001.0)
    EQ_fieldname : Type-available CSV , Regex (e.g., EQ_example0:a=15^int, EQ_example0:a=string, EQ_example1:c=20.5^double, EQ_example2:f=2013-06-08T23:58:56.591%2B09:00^time and long/boolean are available)
    (GT|GE|LT|LE)_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EQ_INNER_fieldname : Type-available CSV , Regex
    (GT|GE|LT|GE)_INNER_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EQ_ILMD_fieldname : Type-available CSV , Regex
    (GT|GE|LT|LE)_ILMD_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EQ_INNER_ILMD_fieldname : Type-available CSV , Regex
    (GT|GE|LT|GE)_INNER_ILMD_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EQ_ERROR_DECLARATION_fieldname : Type-available CSV , Regex
    (GT|GE|LT|LE)_ERROR_DECLARATION_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EQ_INNER_ERROR_DECLARATION_fieldname : Type-available CSV , Regex
    (GT|GE|LT|GE)_INNER_ERROR_DECLARATION_fieldname : Type-available CSV for (int|long|double|boolean|float|time)
    EXISTS_fieldname : true or false (e.g., EXISTS_example:myField=false)
    EXISTS_ILMD_fieldname : true or false (e.g., EXISTS_ILMD_example:myField=false)
    EXISTS_errorDeclaration : true or false
    EQ_eventID : user-defined eventID, auto-generated Mongo object ID (e.g., 57294415deab32126d209d13)
    EQ_errorReason : CSV, Regex (e.g., urn:epcglobal:cbv:error:add)
    EQ_correctiveEventID : CSV, Regex (e.g., 5722d7e1deab322596705146^regex )
    orderBy : (eventTime|recordTime|fieldName)
    orderDirection : ASC, DESC (default)
    eventCountLimit : return specified number of events
    maxEventCount : yield error if # of events exceeds specified number

    {queryName}==SimpleMasterDataQuery

    vocabularyName : Comma separated vocabularyName
    includeAttributes : whether including attributes or not
    includeChildren : whether including children or not
    attributeNames : Comma separated attribute names, result only will include specified list of names
    (EQ|WD)_name : Comma separated attribute names
    HASATTR : Comma separated attribute names
    maxElementCount : yield error if # of events exceeds specified number
    format : (XML|JSON) formatted Output
