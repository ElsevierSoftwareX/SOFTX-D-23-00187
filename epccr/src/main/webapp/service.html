EPCCR RESTful Web Service

Common URL [RESOURCE] : http://{base-url}:{base-port}/epccr/Service/resource
Warning! According to RESTful principle, resource should be represented by URL (e.g. [/resource/urn:epc:id:sgtin:111111111111.1.1]).
However, unfortunately EPC cannot be inserted to URL, thus we insert it as a parameter.
Method Name Description

HTTP.PUT
  • This service allows you to store Context into backend storage
  • Parameter:
    (Mandatory) String epc : epc
    (Optional) Long eventTime : System Time Millisecond when event occurred (Default: CurrentTime)
    (Optional) Long finishTime : System Time Millisecond when event finished (Default: CurrentTime)
    (Optional)You can save any context values such as Temperature, Humidity as you want

  • Example 1 : Store Temperature with default time


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&Temperature=36


    Example 2 : Store Temperature with specified time


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&eventTime=1413541081698&Temperature=36


    Example 3 : Store two temperature within specified time range


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&eventTime=1413541081698&finishTime=1413541081700&Temperature=36,35.8


    Example 4 : Store multiple context


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&Temperature=36&Humidity=100&LightLevel=30

HTTP.GET
  • This service allows you to query Context from backend storage
  • Parameter:
    (Mandatory) String epc : epc
    (Optional) String from : Start Time filter ( Relative Time | Absolute Time )
    (Optional) String until : Start Time filter ( Relative Time | Absolute Time )
    c.f. Time description:
    [Relative Time] relative time from present
    Type: -(s|min|h|d|w|y)
    [Absolute Time] absolute time with format yyyy-MM-dd'T'HH:mm:ss

  • Example 1 : Get all context of specified EPC


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1


    Example 2 : Get context of specified EPC from 100 minutes before


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&from=-100min


    Example 3 : Get context of specified EPC from 100 minutes before to 30 minutes before


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&from=-100min&until=-30min


    Example 4 : Get context of specific EPC with absolute time range


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&from=2014-10-18T10:00:00&until=2015-10-18T10:00:00

HTTP.POST
  • This service allows you to subscribe specific EPC (Real-time Push)
  • Parameter:
    (Mandatory) String epc : epc
    (Mandatory) String destURL : URL to get callback (POST)

  • Example : Subscribe specific EPC


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&destURL=http://localhost/epccr/Test

HTTP.DELETE
  • This service allows you to delete subscription to specific EPC
  • Parameter:
    (Mandatory) String epc : epc
    (Mandatory) String destURL : URL to get callback (POST)

  • Example : Unsubscribe specific EPC


    [RESOURCE]?epc=urn:epc:id:sgtin:111111111111.1.1&destURL=http://localhost/epccr/Test
